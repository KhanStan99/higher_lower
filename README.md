#### Welcome to CarWise Frontend Repository

**Framewoks:**

* [ReactJS](https://reactjs.org/)
* [Material-UI](http://material-ui.com/)

**Languages:**

* JavaScript
* HTML/CSS

**How to run?**

* Clone this repository.
* `cd car-wise-ui`
* `npm install`
* `npm start`
* The project will default load on port 3000
